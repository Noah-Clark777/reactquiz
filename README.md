React Quiz – Interactive Learning App

An interactive quiz application designed to test and reinforce knowledge of React concepts through a dynamic, state-driven user experience.
This project focuses on state management, conditional rendering, and user interaction flows within a single-page React application.

Features:
* Multiple-choice React quiz questions
* Real-time progress tracking
* Score calculation and final results screen
* Dynamic question navigation
* Immediate feedback based on user input
* Responsive design for different devices


Tech Stack:
* Frontend: React
* State Management: useState / useReducer
* Styling: CSS
* Tooling: Vite / Create React App
* Deployment: Netlify



Architecture & Key Concepts:
* State-Driven UI:
The application UI updates dynamically based on quiz progress and user answers.
* Reducer Pattern:
Centralized state transitions for predictable quiz flow (start → answer → next → results).
* Conditional Rendering:
Displays different screens (start, questions, results) based on app state.
* Event Handling:
User interactions (selecting answers, progressing questions) drive application logic.
* Component-Based Design:
Modular components for questions, answers, and results improve maintainability.
